betta (current focus)

GitHub | [Live demo coming...]

A social accountability app where users create habit-tracking challenges and friends can bet on whether they'll succeed. Combines personal goal-setting with friendly competition and social motivation.

Key Features:

  • Create custom habit challenges (exercise routines, study goals, etc.)
  • Friends can place bets on your success or failure
  • Track progress and see real-time updates
  • Cross-platform: responsive web app + native mobile experience

Tech Stack: React and React Router for the frontend, Spring Boot + Hibernate with PostgreSQL for the backend

What I'm learning: Taking an app from concept to production: system architecture, REST API design, state management, betting/scoring logic, habit tracking UX and deployment workflows.
