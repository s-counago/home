Realista (foco principal)

GitHub | [Live demo coming...]

Buscas casa? Con realista es más fácil. Miles de reviews reales sobre caseros, inmobiliarias y sus viviendas en alquiler.

Funciones principales:

  • Permite leer reviews reales, de gente que con experiencia de primera mano habitando en viviendas y lidiando con inmobiliarias.
  • Crear reviews sobre los pisos donde se vivió.
  • Multiplataforma: web app responsiva + experiencia android nativa (WIP).

Tech Stack: React y React Router para el frontend, Spring Boot + Hibernate con PostgreSQL para el backend.

Lo que estoy aprendiendo: a cómo lidiar con lógicas de verificación complejas (para reviews), a crear buenas experiencias de usuario (y lo difícil que es) y a gestionar información del mundo real, intentando incluir todas sus aristas.

betta

GitHub | [Live demo coming...]

Una app para mantenerte firme con los hábitos que te propones conseguir y mantener en el tiempo donde quien tú elijas puede apostar por ti (o contra ti!). Monetiza a tus amigos más haters apostando por ti (literalmente).

Funciones principales:

  • Crea desafíos propios y personalizados (ejercicio, estudios, lectura, socialización...)
  • Tus amigos pueden apostar en tu éxito o fracaso.
  • Controla el progreso de cada hábitoen tiempo real.
  • Multiplataforma: web app responsiva + experiencia android nativa.

Tech Stack: React y React Router para el frontend, Spring Boot + Hibernate con PostgreSQL para el backend.

Lo que estoy aprendiendo: A llevar una app desde su idea y concepto a producción: desde la arquitectura del sistema, el diseño de la API RESTful, gestión del state, lógica de las apuestas, desarrollo y diseño de UX y flujos de despliegue.

Bot de cobertura en contratos futuros perpetuos de posiciones de provisión de liquidez UniV3

GitHub

Un bot automático que cubre la exposición generada a la volatilidad de activos arbitrarios que intenta minimizar la pérdida impermanente inherente a las posiciones UniV3. Deprecado por no ser rentable, tristemente :(

Funciones principales:

  • Cobertura automágica en base a la exposición en largo del LP
  • Integración con Hyperliquid para gestionar el posicionamiento de los contratos perpetuos

Tech Stack: Python y Solidity

Lo que aprendí: Cómo aplicar matemáticas discretas (para comprender cómo analizar posiciones UniV3) y que el dinero no crece en los árboles :(
